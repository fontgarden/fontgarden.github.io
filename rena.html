About Rena

Rena is a neo-grotesk grid based typeface designed by Eli Heuer in 2024. It take inspiration from grid-based typefaces like Replica, but uses a higer resoltion 4 unit grid for better optical adjustments. This grid based system has many aditional advantages, the make it easier for contributors to make pull requests, improvments, and language extensions without working closly with the original designer, and it allowes for a powers-of-2 based design system that helps LLMs to more effectily train on and augment the original design. It's all just compression and interpolation.
